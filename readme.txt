# Pushup Detector

There are two versions here,
detect_AI.py uses a SVC model that I trained on around 13,000 images. (94% Accuracy)
detect_manial.py calculates the wrist distance to infer if a pushup is being done.

## Prerequisites

- Python 3.7 or higher
- Pip package manager

## Installation

Install the required Python packages using the following command:

pip install -r requirements.txt

## Run the Programs

>python detect_AI.py 
>python detect_manual.py
